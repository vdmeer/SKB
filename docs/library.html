SKB Library

I do maintain my own research library. Essentially published (and some not formally published) items. The library is part of my SKB project on Github. I host the compiled version on GH pages (SKB project). There are two main compilations: the complete library and my own publications. Everything else is grouped mainly by BibLaTeX entry types. The GH pages files are self-contained HTML documents.

Main Groups

Complete library All items I have in my library, frequently updated when I add items or fix errors. GH pages source
My own publications Complete set of my own publications. GH pages source
